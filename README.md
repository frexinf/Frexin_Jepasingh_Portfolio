# Frexin Jepasingh — Portfolio Website

Personal portfolio website for Frexin Jepasingh, IoT & Network Engineer | IT Support Professional.

## 🚀 Live Site

Deployed via Netlify: `https://frexinjepasingh.netlify.app`

## 📁 Project Structure

```
frexin-portfolio/
├── index.html       # Complete single-page portfolio
├── netlify.toml     # Netlify deployment config
├── .gitignore       # Git ignore rules
└── README.md        # This file
```

## 🛠️ Tech Stack

- Pure HTML, CSS, JavaScript — no frameworks
- Google Fonts (Syne + DM Sans)
- CSS keyframe animations & IntersectionObserver
- Canvas-based particle background

## 📦 Deploy to Netlify via GitHub

1. Push this repo to GitHub
2. Go to [netlify.com](https://netlify.com) → **Add new site → Import from Git**
3. Connect your GitHub account and select this repo
4. Build settings are auto-detected via `netlify.toml`
5. Click **Deploy** — your site goes live instantly

## ✏️ Customization

All content is in `index.html`. To update:
- **Skills / Projects / Experience** — edit the JS data arrays near the top of the `<script>` block
- **Colors** — change CSS variables in `:root { ... }`
- **Resume PDF** — update the `downloadResume()` function content
